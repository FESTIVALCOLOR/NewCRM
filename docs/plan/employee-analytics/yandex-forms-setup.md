## 1. Подготовка

### Шаг 1.1: Откройте Яндекс Формы

1. Откройте браузер
2. Перейдите на сайт: **https://forms.yandex.ru**
3. Войдите под тем же аккаунтом Яндекс, что используется для Яндекс.Диска в CRM
4. Вы увидите главную страницу «Мои формы»

### Шаг 1.2: Убедитесь, что у вас бизнес-аккаунт

- Если вы используете Яндекс 360 для бизнеса — webhook (отправка ответов на наш сервер) доступна
- Если обычный аккаунт — webhook недоступен, но есть альтернатива: мы будем забирать ответы через API Яндекс Форм или CSV-экспорт
- В любом случае **создание формы одинаковое** — различается только способ получения ответов

## 2. Создание формы для индивидуальных проектов

### Шаг 2.1: Создайте новую форму

1. На главной странице Яндекс Форм нажмите кнопку **«Создать форму»** (жёлтая кнопка вверху)
2. Выберите **«Создать с нуля»** (не шаблон)
3. Вверху страницы кликните на название «Новая форма» и введите: **«Оценка качества — Индивидуальный проект»**

### Шаг 2.2: Настройте описание формы

1. Под названием формы есть поле «Описание» — нажмите на него
2. Введите текст:

```
Уважаемый клиент!

Благодарим вас за сотрудничество с Interior Studio.
Мы будем признательны, если вы уделите 2 минуты и оцените нашу работу.
Ваши ответы помогут нам стать лучше.

Все ответы анонимны и используются только для улучшения качества наших услуг.
```

### Шаг 2.3: Добавьте Вопрос 1 — NPS (шкала 0-10)

1. Нажмите кнопку **«+ Добавить вопрос»**
2. Выберите тип вопроса: **«Оценка по шкале»** (иконка со звёздочками/ползунком)
3. Заполните:
   - **Текст вопроса:** `Насколько вы готовы рекомендовать нашу студию друзьям и знакомым?`
   - **Тип шкалы:** Числовая шкала (или ползунок)
   - **Минимум:** `0`
   - **Максимум:** `10`
   - **Подпись минимума:** `Точно не буду рекомендовать`
   - **Подпись максимума:** `Обязательно порекомендую`
4. Включите переключатель **«Обязательный вопрос»** (справа)
5. В расширенных настройках (иконка шестерёнки) задайте **ID вопроса:** `nps_score`
   - Если нет возможности задать ID — запомните порядковый номер вопроса (1)

### Шаг 2.4: Добавьте Вопрос 2 — Общая удовлетворённость (1-5)

1. Нажмите **«+ Добавить вопрос»**
2. Тип: **«Оценка по шкале»**
3. Заполните:
   - **Текст вопроса:** `Насколько вы довольны результатом работы в целом?`
   - **Тип шкалы:** Звёзды (если есть) или числовая 1-5
   - **Минимум:** `1` (Очень недоволен)
   - **Максимум:** `5` (Очень доволен)
4. **Обязательный:** Да
5. **ID вопроса:** `csat_score`

### Шаг 2.5: Добавьте Вопрос 3 — Качество дизайна (1-5)

1. **«+ Добавить вопрос»** → **«Оценка по шкале»**
2. Заполните:
   - **Текст:** `Оцените качество дизайн-проекта`
   - **Шкала:** 1-5
   - **Подпись 1:** `Очень плохо`
   - **Подпись 5:** `Отлично`
3. **Обязательный:** Да
4. **ID:** `design_score`

### Шаг 2.6: Добавьте Вопрос 4 — Соблюдение сроков (1-5)

1. **«+ Добавить вопрос»** → **«Оценка по шкале»**
2. Заполните:
   - **Текст:** `Насколько хорошо соблюдались оговорённые сроки?`
   - **Шкала:** 1-5
   - **Подпись 1:** `Постоянные задержки`
   - **Подпись 5:** `Всё точно в срок`
3. **Обязательный:** Да
4. **ID:** `deadline_score`

### Шаг 2.7: Добавьте Вопрос 5 — Коммуникация (1-5)

1. **«+ Добавить вопрос»** → **«Оценка по шкале»**
2. Заполните:
   - **Текст:** `Оцените качество общения с командой (информирование, отзывчивость, вежливость)`
   - **Шкала:** 1-5
3. **Обязательный:** Да
4. **ID:** `communication_score`

### Шаг 2.8: Добавьте Вопрос 6 — Соответствие ожиданиям (1-5)

1. **«+ Добавить вопрос»** → **«Оценка по шкале»**
2. Заполните:
   - **Текст:** `Насколько результат соответствует вашим ожиданиям и техническому заданию?`
   - **Шкала:** 1-5
3. **Обязательный:** Да
4. **ID:** `expectations_score`

### Шаг 2.9: Добавьте Вопрос 7 — Свободный комментарий

1. **«+ Добавить вопрос»** → **«Длинный текст»** (тип Textarea)
2. Заполните:
   - **Текст:** `Что бы вы хотели отметить или предложить улучшить?`
   - **Подсказка в поле:** `Необязательно. Максимум 2000 символов.`
3. **Обязательный:** Нет (выключен)
4. **ID:** `comment`

### Шаг 2.10: Добавьте скрытые поля

Скрытые поля не видны клиенту, но передаются через URL. Это нужно для привязки ответа к договору.

1. Нажмите **«+ Добавить вопрос»** → **«Короткий текст»**
2. Заполните:
   - **Текст:** `contract_id`
   - **Обязательный:** Нет
3. **Важно:** Сделайте это поле **скрытым**:
   - В настройках вопроса (шестерёнка) найдите опцию **«Скрытое поле»** или **«Предзаполнение из URL»**
   - Включите её
   - Укажите **имя параметра URL:** `contract_id`
4. Повторите для ещё двух скрытых полей:

   **Поле 2:**
   - Текст: `project_type`
   - Параметр URL: `project_type`

   **Поле 3:**
   - Текст: `survey_token`
   - Параметр URL: `survey_token`

> **Если в вашей версии Яндекс Форм нет «скрытых полей»:**
> Альтернатива — добавить обычные текстовые поля с этими именами, но поставить галочку «Предзаполнение из URL-параметров». Клиент увидит их заполненными, но не сможет изменить если добавить атрибут «Только для чтения».
> Ещё один вариант — не добавлять скрытые поля, а использовать параметры в URL формы. CRM будет сопоставлять ответы по времени + contract_number.

### Шаг 2.11: Настройте страницу «Спасибо»

1. Перейдите в раздел **«Настройки»** (шестерёнка вверху справа)
2. Найдите **«Текст после отправки»** или **«Страница благодарности»**
3. Введите:

```
Спасибо за ваш отзыв!

Ваше мнение очень важно для нас и поможет улучшить качество наших услуг.

С уважением,
Команда Interior Studio
```

### Шаг 2.12: Опубликуйте форму

1. Нажмите кнопку **«Опубликовать»** (или «Поделиться»)
2. Скопируйте **ссылку на форму** — она будет выглядеть примерно так:
   `https://forms.yandex.ru/u/XXXXXXXXX/`
3. **Запишите эту ссылку** — она понадобится для настройки CRM

## 3. Создание формы для шаблонных проектов

### Самый быстрый способ — скопировать форму

1. На главной странице Яндекс Форм найдите только что созданную форму **«Оценка качества — Индивидуальный проект»**
2. Нажмите на **три точки** (⋮) справа от неё
3. Выберите **«Копировать»** (или «Дублировать»)
4. Откройте копию

### Что изменить в копии

1. **Название:** Измените на **«Оценка качества — Шаблонный проект»**
2. **Все вопросы остаются одинаковыми** — шаблонные проекты оцениваются по тем же критериям
3. **Опубликуйте** и **скопируйте ссылку**

> Форма для шаблонных проектов идентична индивидуальной. Разделение нужно для раздельной статистики в CRM — CRM будет отправлять разные ссылки в зависимости от типа проекта.

## 4. Создание формы для авторского надзора

### Скопируйте форму индивидуального проекта

1. Аналогично шагу 3 — скопируйте форму
2. Откройте копию

### Что изменить

1. **Название:** **«Оценка качества — Авторский надзор»**

2. **Описание** — замените на:
```
Уважаемый клиент!

Благодарим вас за сотрудничество с Interior Studio.
Мы будем признательны, если вы уделите 2 минуты и оцените качество
авторского надзора на вашем объекте.
```

3. **Добавьте новый вопрос** (перед свободным комментарием):
   - Нажмите **«+ Добавить вопрос»** → **«Оценка по шкале»**
   - **Текст:** `Оцените качество авторского надзора (контроль строительных работ, закупок, визиты на объект)`
   - **Шкала:** 1-5
   - **Обязательный:** Да
   - **ID:** `supervision_score`
   - **Перетащите** этот вопрос выше свободного комментария (вопрос 7 из 8)

4. **Опубликуйте** и **скопируйте ссылку**

## 5. Настройка webhook

> **Webhook** — это автоматическая отправка ответов на наш сервер при каждом заполнении формы.
> Доступен в Яндекс 360 для бизнеса. Если у вас личный аккаунт — перейдите к разделу 5Б.

### 5А. Настройка webhook (Яндекс 360 для бизнеса)

Повторите эти шаги **для каждой из трёх форм:**

1. Откройте форму в редакторе
2. Перейдите в **«Настройки»** → **«Интеграции»** (или **«Уведомления»**)
3. Найдите раздел **«Отправлять ответы по HTTP»** (HTTP-уведомления / Webhook)
4. Включите его
5. Заполните:
   - **URL:** `https://ВАШ_ДОМЕН/api/surveys/yandex-webhook`
     (замените `ВАШ_ДОМЕН` на адрес вашего CRM сервера, например `crm.interiorstudio.ru`)
   - **Метод:** POST
   - **Формат:** JSON
6. Нажмите **«Сохранить»**

> **Где взять URL сервера?**
> Это тот же домен, по которому работает ваш CRM API. Если CRM доступна по адресу
> `https://crm.example.com`, то webhook URL будет `https://crm.example.com/api/surveys/yandex-webhook`.
> Уточните у разработчика, если не уверены.

### 5Б. Альтернатива без webhook (личный аккаунт)

Если webhook недоступен, есть два альтернативных способа получать ответы:

**Способ 1: API Яндекс Форм (автоматический)**

CRM будет сама периодически запрашивать новые ответы через API Яндекс Форм. Для этого:

1. Перейдите на https://oauth.yandex.ru/client/new
2. Создайте приложение с доступом к Яндекс Формам
3. Получите OAuth-токен
4. Передайте токен разработчику

**Способ 2: CSV-экспорт (ручной)**

1. Раз в неделю заходите в каждую форму
2. Нажимайте **«Скачать результаты»** → **CSV**
3. Загружайте файл в CRM (будет специальная кнопка «Импорт опросов»)

> Рекомендация: если нет бизнес-аккаунта, лучше всего **Способ 1** (API). Разработчик настроит автоматический опрос раз в час.

## 6. Тестирование

После создания всех трёх форм обязательно протестируйте каждую:

### Шаг 6.1: Проверьте скрытые поля

1. Скопируйте ссылку на форму, например:
   `https://forms.yandex.ru/u/XXXXXXX/`
2. Добавьте параметры в конец ссылки:
   `https://forms.yandex.ru/u/XXXXXXX/?contract_id=9999&project_type=test&survey_token=test123`
3. Откройте эту ссылку в браузере
4. Убедитесь что:
   - Форма открывается нормально
   - Скрытые поля **не видны** клиенту (или видны, но предзаполнены)
   - Все вопросы отображаются корректно

### Шаг 6.2: Заполните тестовый ответ

1. Ответьте на все вопросы (любые значения)
2. Нажмите «Отправить»
3. Убедитесь, что видите страницу «Спасибо»

### Шаг 6.3: Проверьте получение ответа

1. Вернитесь в редактор формы
2. Перейдите на вкладку **«Ответы»**
3. Убедитесь, что ваш тестовый ответ появился
4. Проверьте, что скрытые поля (contract_id, project_type, survey_token) записались со значениями из URL

### Шаг 6.4: Проверьте webhook (если настроили)

1. После отправки тестового ответа, проверьте логи сервера
2. Должен быть входящий POST-запрос на `/api/surveys/yandex-webhook`
3. Если webhook не срабатывает — проверьте URL и доступность сервера извне

## Часто задаваемые вопросы

### Сколько раз клиент может заполнить форму?

По умолчанию Яндекс Формы разрешают повторное заполнение. CRM на своей стороне игнорирует повторные ответы — учитывается только первый ответ с данным `survey_token`.

### Можно ли изменить вопросы после публикации?

Да, но осторожно:
- **Можно:** менять текст вопросов, описание, страницу «Спасибо»
- **Нельзя:** удалять вопросы или менять их порядок (это сломает маппинг ответов)
- **Можно:** добавлять новые вопросы в конец формы

### Что если webhook не работает?

1. Проверьте, что URL сервера доступен из интернета (не localhost)
2. Проверьте, что URL начинается с `https://` (не http)
3. Проверьте логи сервера на наличие ошибок
4. Как временное решение — используйте CSV-экспорт

### Нужно ли платить за Яндекс Формы?

- **Базовый функционал** (создание форм, сбор ответов) — бесплатно
- **Webhook** — доступен в Яндекс 360 для бизнеса (платная подписка)
- **API** — доступен в Яндекс 360 для бизнеса
- **CSV-экспорт** — бесплатно (но ручной)

### Могу ли я посмотреть статистику ответов в Яндекс Формах?

Да! В каждой форме есть вкладка «Ответы» → «Статистика». Там видны:
- Общее количество ответов
- Среднее значение по каждому вопросу
- Графики распределения
- Отдельные ответы

Но основная аналитика будет в CRM — там ответы привязаны к конкретным сотрудникам и проектам.
